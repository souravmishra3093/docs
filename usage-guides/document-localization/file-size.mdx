## File Size Limits by Extension

**10 MB**
- `.docx`, `.pptx`, `.xlsx`, `.pdf`, `.odt`, `.ott`, `.ods`, `.ots`, `.odp`, `.otp`

**5 MB**
- `.html`, `.htm`, `.xhtml`, `.ttx`, `.mif`, `.idml`, `.icml`, `.dita`, `.tsv`, `.xml`, `.dtd`, `.json`, `.yaml`, `.properties`, `.resx`, `.strings`, `.srt`, `.wix`

**1 MB**
- `.txt`, `.csv`


## Max Word Count
**10,000 words (Ten Thousand Words)**
