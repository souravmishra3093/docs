# Welcome to Reverie API Documentation

Welcome to your go-to resource for integrating powerful language technologies into your applications.
Our suite of APIs is designed to help developers like you streamline text and voice processing with ease.
Whether you're building a cutting-edge app or enhancing an existing solution, our APIs offer the flexibility and power you need.

## Key Features

<CardGroup cols={2}>
  <Card title="Translate Text" icon="pen-to-square">
    Use the `/translate` endpoint to translate text from one language to
    another. Supports 10 Indic languages along with English, achieving
    best-in-class performance. **[Know more]**
  </Card>
  <Card title="Speech to Text" icon="pen-to-square">
    Use the `/speech-to-text` endpoint to convert spoken language into written
    text. Output is returned in the same input language. **[Know more]**
  </Card>
</CardGroup>

<CardGroup cols={2}>
  <Card title="Speech to Text Translate" icon="pen-to-square">
    Use the `/speech-to-text-translate` endpoint to combine speech recognition
    and translation, allowing you to convert spoken language directly into
    translated text. **[Know more]**
  </Card>
  <Card title="Text to Speech" icon="pen-to-square">
    Use the `/text-to-speech` endpoint to convert written text into spoken
    words. Supports natural-sounding voices across 10 languages. **[Know more]**
  </Card>
</CardGroup>

<CardGroup cols={2}>
  <Card title="Call Analytics" icon="pen-to-square">
    Use the `/call-analytics` endpoint to perform intelligent question-answering
    on recorded calls or conversations. **[Know more]**
  </Card>
  <Card title="Text Analytics" icon="pen-to-square">
    Use the `/text-analytics` endpoint to conduct advanced question-answering on
    written text. **[Know more]**
  </Card>
</CardGroup>

## Getting Started

To get started with Reverie APIs, follow these steps:

1. **Authenticate**: Learn how to authenticate your API requests in the Authentication Guide.

2. **Meta Prompt**: You can now use our Meta Prompt directly to guide any AI chat model with the context needed to use Sarvam’s APIs effectively. An example on AI Studio with Gemini’s latest model: Explore here

3. **Try Examples**: Use the Usage Guides to see examples and best practices.
